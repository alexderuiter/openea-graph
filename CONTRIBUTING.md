# Contributing

The project is specification-first.

Proposed semantic changes should be documented in an RFC before changing the core model. Small clarifications and examples may be submitted directly.

Do not copy proprietary specification text into this repository. Metamodel definitions must be independently expressed and checked against applicable licensing conditions.

## Licensing contributions

Unless explicitly stated otherwise, contributions intentionally submitted for
inclusion in OpenEA Graph are provided under the Apache License 2.0, as described
in section 5 of that license. Only submit work that you have the right to
contribute. In particular, check whether an employer or client owns work created
as part of your employment or an engagement.
