# RFC-0001 — Foundational decisions

Status: Accepted

## Decisions

1. The core is a generic graph rather than ArchiMate-specific.
2. ArchiMate is implemented as a metamodel package.
3. All first-class objects have URIs.
4. The canonical repository stores explicit facts.
5. Reasoning is modular and derived assertions remain distinguishable.
6. AI and spreadsheet imports produce reviewable change sets.
7. Plain JSON is the initial interchange format.
8. RDF/JSON-LD compatibility is a planned export and mapping layer.
